Pasos para decodificar el mensaje:
  1. Leer mensaje.
  2. Identificar la posición de las letras en el abecedario.
  3. Recorrer una posición a la izquierda (1 valor menor).
  4. Guardar la letra de esa posición y escribirla.
  5. Repetir con la cantidad de letras que tenga el mensaje que se quiere decodificar.
  6. Juntar las letras y ver el mensaje final.
